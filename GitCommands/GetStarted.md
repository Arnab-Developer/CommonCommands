# Get started

Initialize local git

```
git init
```

Show changes of working folder

```
git status
```

Stage file to commit

```
git add [file path]
```

Stage all files to commit

```
git add .
```

Commit staged files

```
git commit -m "[Commit message]"
```

Multi-line commit message (it will open an external editor for commit messsage)

```
git commit
```
